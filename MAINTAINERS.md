## Becoming a maintainer

Zynax is currently a **single-maintainer project actively looking for
co-maintainers**. Sustained, high-quality contributions — code, reviews,
documentation, issue triage — are the path in; see
[GOVERNANCE.md §10](GOVERNANCE.md#10-adding-and-removing-maintainers) for the
formal process and [CONTRIBUTING.md](CONTRIBUTING.md) for how to start.
Issues labeled `good first issue` are curated entry points with complete
acceptance criteria.
