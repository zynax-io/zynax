# Product

Product strategy, positioning, adoption, and sustainability research for Zynax.

This folder holds the **product-facing** view of the project — who it is for, how it wins,
how it gets adopted, and how it sustains itself — as a complement to the engineering and
architecture docs. Strategic claims here are grounded in the repository's own decision
record and live milestone state, with the "real vs aspirational" line drawn explicitly.
